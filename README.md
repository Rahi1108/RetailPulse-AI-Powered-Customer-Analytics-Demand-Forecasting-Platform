# RetailPulse: AI-Powered Customer Analytics & Demand Forecasting Platform

RetailPulse is a multi-page analytics dashboard built with Python and Streamlit that helps retail businesses analyze customer behavior, forecast demand, identify churn risks, and optimize inventory decisions.

## Features

### Exploratory Data Analysis (EDA)

* Revenue and sales performance tracking
* Customer and transaction analysis
* Interactive visualizations

### Customer Segmentation

* RFM-based customer profiling
* Behavioral customer groups
* Marketing-focused insights

### Demand Forecasting

* Future sales trend prediction
* Demand planning support

### Churn Analysis

* Customer inactivity detection
* Risk categorization
* Retention recommendations

### Inventory Optimization

* Stock monitoring
* Reorder recommendations
* Inventory health assessment

## Technology Stack

* Python
* Streamlit
* Pandas
* NumPy
* Plotly
* Scikit-Learn
* Git & GitHub
